## Features ✨

- 📥 Download list video of a user
- 🚫 Remove TikTok watermark automatically
- 📁 Import multiple URLs from text files
- 📊 Download progress tracking

## Requirements 📋

- .NET 8.0 Runtime
- Chrome Browser (for WebDriver)
